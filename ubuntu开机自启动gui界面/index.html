ubuntu开机自启动GUI界面

1.不需要GUI的后台程序只需输入

1

sudo gedit /etc/rc.local

在exit 0 前添加一行

1

cd /home/lin/; ./run_guard.sh &

即可正常开启启动程序

2.开机启动GUI界面程序

由于GUI界面需要在xserver服务启动之后才能启动，放在rc.local中会调用失败
XServer启动的时候会调用 /etc/X11/Xsession.d/55gnome-session_gnomerc 该脚本的内容就是检测 $HOME/.gnomerc是否存在，如果存在就调用
所以:

gedit ~/.gnomerc

放入下列脚本即可

1
2
3
4
5

#!/bin/bash

cd /home/lin/

./run_guard.sh &

程序不休眠

在桌面右上角设置的不挂起，要在桌面加载之后才会生效，当我们无桌面或将桌面禁用就可以用到下列方法
在/etc/profile中添加

1
2
3

export DISPLAY=:0
xset -dpms
xset s 0
